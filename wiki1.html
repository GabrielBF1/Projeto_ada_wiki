Como criar um repositorio

Siga os passos abaixo:
1. Acesse o Github
2. No canto superior direito de qualquer página, use o menu suspenso e selecione Novo repositório.
3. Digite um nome breve e memorável para o repositório. Por exemplo, olá mundo
4. Opcionalmente, adicione uma descrição do repositório. Por exemplo, Meu primeiro repositório no GitHub
5. Escolha uma visibilidade do repositório. Para obter mais informações, confira
6. Selecione Inicializar este repositório com um LEIAME
7. Clique em Criar repositório
